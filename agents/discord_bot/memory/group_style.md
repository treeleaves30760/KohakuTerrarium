# Group Communication Style

This file stores observed communication patterns from the group.
Update this as you learn how the group communicates.

## Vocabulary & Slang

- **換日 (Daily Reset)**: 每天下午四點（16:00）左右會有人發出的訊息，推測是某種遊戲簽到、伺服器習慣。
- **~ask**: 一種問答指令，由「栞」機器人回應肯定或否定。
- **跑**: 指跑團（TRPG 遊戲過程）。
- **難繃**: 難以繃住（情緒/笑意），指看到令人哭笑不得或極度荒謬的事情。
- **條碼/OD**: 涉及自殘或藥物過量的術語（出現在柴修轉發的投稿文中，需注意但不主動提及）。

## Typical Message Patterns

- 成員喜歡發送貼圖和 GIF。
- KBlueLeaf 會發送伺服器狀態更新。
- 每天固定有人發送「換日」。
- **柴修會連續發送長篇的定型文（Copypasta）或轉載笑話。**
- **成員對於這種刷屏行為已經習以為常，多以貼圖或簡短吐槽（如「難繃」）回應。**
- **群組活躍於 TRPG 交流與遊戲討論。**

## Tone & Formality

- 輕鬆、非正式，充滿 ACG、TRPG 與技術討論氛圍。

## Language Usage

- 主要使用繁體中文。
- 夾雜技術術語（如 Python, Framework）與遊戲/跑團術語。
